TAGs em HTML

Em HTML5, podemos inserir os elementros da lista a seguir. Passe o mouse sobre os termos para descobrir qual a TAG deverpa utilizar me cada caso.

Para isso são utilizados os caracteres 'maior que' e 'menor que'. São representados pelos caracteres < > & l t / & g t

A TAG que apresenta o texto ao passa o mouse em cima do texto, como um HINt é TAG abbr.
Esta TAG significa abreviação.

Exemplos

  • Favicon
  • Parágrafo
  • Abreviação
  • Título de Página
  • Imagens
  • Listas numeradas
  • Listas não ordenadas
  • Item de Listas
  • Lista de Termos
  • Termo
  • Definição do termo
