TAGs em HTML

Em HTML 5 podemos inserir os elementos da lista a seguir. Passe o mouse osbre os termos para descobrir qual a TAG deverá utilizar em cada caso.

Para isso são utilidos os caracteres menor que '<' e maior que '>'. São representados pelos caracteres & l t / & g t.

A TAG que apresenta um hint ao passar o mouse por cima do texto, é a TAG <abbr>
Essa TAG significa abreviação

Exemplos

  • Favicon
  • Parágrafo
  • Abreviação
  • Heading da página
  • Imagens
  • Listas ordenadas
  • Listas não ordenadas
  • Item da Lista
  • Lista de Termos
  • Nome do Termo
  • Definição do Termo
